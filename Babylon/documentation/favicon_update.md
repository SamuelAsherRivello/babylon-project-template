# Babylon Project Template

This project uses the Babylon.js favicon for its page icon.

Babylon.js favicon source: https://www.babylonjs.com/favicon.ico

To update the favicon, the following steps were taken:
- Downloaded the Babylon.js favicon.ico
- Updated index.html to reference the new favicon

---

If you need to update the favicon again, replace the favicon.ico in the project root and update the <link rel="icon"> in index.html if needed.
